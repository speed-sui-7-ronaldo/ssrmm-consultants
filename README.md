# SSRMM Consultants (OPC) Pvt. Ltd.
## Finance Outsourcing Excellence — Official Website

Built with **React 18 + Vite + Recharts**

---

## 🚀 Quick Start

### Prerequisites
- Node.js 18+ installed
- npm or yarn

### Install & Run

```bash
# 1. Install dependencies
npm install

# 2. Start development server
npm run dev

# Open http://localhost:3000
```

### Build for Production

```bash
npm run build
npm run preview
```

## 🎨 Design System

| Token       | Value      | Usage                  |
|-------------|------------|------------------------|
| Deep Olive  | `#1e2d07`  | Footer, dark sections  |
| Olive       | `#3a5212`  | Primary buttons, headings |
| Mid Olive   | `#4e6e18`  | Hover states, accents  |
| Sage Green  | `#8eb530`  | CTAs, highlights       |

## 📋 Sections

1. **Navbar** — Sticky, transparent → deep olive on scroll
2. **Hero** — Full-screen gradient, animated badge, 4 stat cards, dual CTAs
3. **Stats Bar** — Animated counters (scroll-triggered): 194K+, 600+, 45%, 75%
4. **About** — Company story, progress bars (BPO/KPO/ITO mix), certifications
5. **Services** — 3-tab switcher (BPO / KPO / ITO), 6 cards each
6. **Markets** — Bar chart (FSO 2024–2035), Line chart (FAO vs BPO), process share
7. **India's Edge** — 6 advantage cards with animated hover
8. **Global Reach** — 6 regional cards (Americas, APAC, EU, etc.)
9. **Technology** — Dark olive section, 6 tech cards, adoption bar chart
10. **Why SSRMM** — 6 numbered differentiator cards
11. **Contact** — Split layout: info + consultation form with submission state
12. **Footer** — 4-column grid with links
